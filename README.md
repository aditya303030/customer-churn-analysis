# Customer Intelligence Platform

An end-to-end customer analytics and machine learning project using e-commerce data.

## Goal

Build a customer intelligence platform that helps a business understand:

- Customer value
- Revenue trends
- Product performance
- Customer satisfaction
- Delivery performance
- Repeat-purchase and churn risk

## Current Stage

Stage 1: Analyst Version

- Data ingestion
- Exploratory data analysis
- PostgreSQL loading
- SQL-based customer analytics
- Customer 360 table
- Dashboard

## Tech Stack

- Python
- SQL
- PostgreSQL
- Pandas
- Matplotlib/Seaborn
- Streamlit or Power BI
- Scikit-learn
- Git/GitHub

## Dataset

Olist Brazilian E-Commerce Public Dataset from Kaggle.
